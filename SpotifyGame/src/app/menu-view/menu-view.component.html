How to Play the Spotify Game:

  1. Type in an artist whose songs you would like to guess.
  2. Listen to the provided song preview for that artist.
  3. Enter your song guess into the input box below the song preview. Remember to use exact spelling.
  4. Click 'Submit Guess' to see if you got the correct answer!
  5. If you did, move back to step 3 and repeat!
  6. If you did not, you will be taken to a Game Over screen where your scores for that artist are displayed.
  7. Press the button on the Game Over screen to go back to the menu.
  8. Enjoy!
